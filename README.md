## 🚀 Adventure Works – Sales Data Analysis Project  

### 📌 Overview  
This project focuses on analyzing Adventure Works sales data to gain insights into sales performance, profit, production cost, customer trends, product performance, and regional sales.  
The final output is presented through interactive dashboards to support data-driven business decisions.  

---

### 🛠️ Tools Used  
- Excel  
- MySQL  
- Power BI  
- Tableau  

---

### 📂 Data Source & Import  
Sales, product, customer, subcategory, category, date, and region data were imported from Excel files into MySQL using the Table Data Import Wizard.  

---

### 🧹 Data Cleaning  
- Removed duplicates  
- Handled missing (null) values  
- Standardized date formats  
- Validated data consistency  

---

### 🔗 Data Modeling  
- Built relationships between fact (sales) and dimension tables (product, customer, subcategory, category, date, region)  
- Created calculated fields for sales, cost, and profit  

---

### 📊 Key Insights  

#### 💰 Total Sales Performance  
- Total Sales: **29.30M**  
- Highest Year: **2011 (7.07M)**  

#### 📈 Profit & Cost Analysis  
- Total Profit: **12.08M**  
- Production Cost: **17.28M**  

#### 🌍 Regional Insights  
- Top Region: **Australia (9.8M)**  
- Followed by: Southwest, Northwest, United Kingdom  

#### 🛍️ Product & Customer Insights  
- Top Product: **Road Bikes**  
- Top Customer Region: **Australia**  

---

### 📊 Dashboards & Outcome  
Developed interactive dashboards with filters such as **Year, Quarter, Month, Region, Product, and Category** to enable dynamic analysis and support business decision-making.  

---

### 🎯 Conclusion  
This project demonstrates end-to-end data analysis, including data extraction, cleaning, modeling, and visualization to generate actionable business insights.
